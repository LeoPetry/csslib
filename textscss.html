TEXT.SCSS

Awesome semantic classes for text styling

Text.scss is a library of semantic classes you can use to speed up Front-End development.
Our goal is to give you a collection of classes you can use to style text. In our setup file you can redefine the scale of font sizes, set different font families, colors, spacing and everything related to text. In order to do so you must have the sass compiler available in your machine. Don't worry we will guide you through everything.
